A Cryptocurrency Tracker built using [React](https://reactjs.org) and powered by the CoinGecko [API](https://www.coingecko.com/en/api).

## Setup

1. Install the project using npm:

```
$ cd crypto-tracker && npm install
```

2. Start the project:

```
$ npm start
```
